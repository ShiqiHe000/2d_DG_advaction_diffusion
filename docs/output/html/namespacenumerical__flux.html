Detailed Description

Numerical flux is a combination of two fluxes on the interface. Based on the interface is an element interface or a boundary interface, the numerical flux is computed by either two correspond fluxes on the element interfaces, or element interface and boundary condition (we call it external state). On each direction, i.e. x and y, each element has two interfaces. And the target element's neighbor is decided by its dual coordinate in i-j plane.
